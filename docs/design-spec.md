# 设计规范 — 健身计划管理 App

## 风格定位

**清新文艺风** — 温暖、雅致、有书卷气。如一本精心排版的纸质手帐，静谧而有质感。

## 色彩系统（升级版）

| 变量名 | 色值 | 用途 |
|--------|------|------|
| --color-primary | #6B9FD4 | 主色：按钮、Tab图标、强调元素 |
| --color-primary-dark | #4A7FB5 | 深蓝：标题、重点文字 |
| --color-primary-light | #C5D9F0 | 浅蓝：卡片背景、高亮区域 |
| --color-accent | #D4A574 | 暖棕：装饰线、小点缀 |
| --color-accent-light | #F5EDE3 | 暖米：卡片暖色调 |
| --color-bg-start | #EDF3F8 | 背景渐变起点：淡蓝灰 |
| --color-bg-mid | #F5F0EB | 背景渐变中间：暖灰米 |
| --color-bg-end | #F8F6F3 | 背景渐变终点：暖白 |
| --color-card | #FFFFFF | 白色：卡片底色 |
| --color-card-warm | #FEFCF9 | 暖白：特殊卡片 |
| --color-text | #3D3D3D | 深灰：正文文字 |
| --color-text-light | #8B8B8B | 浅灰：辅助文字、时间 |
| --color-danger | #D48686 | 删除/支出按钮 |
| --color-success | #86BDA0 | 完成/收入按钮 |

## 字体

| 层级 | 大小 | 字重 | 用途 |
|------|------|------|------|
| H1 | 20px | 700 | 页面标题 |
| H2 | 17px | 600 | 板块标题 |
| H3 | 15px | 500 | 卡片标题 |
| Body | 14px | 400 | 正文 |
| Caption | 12px | 400 | 辅助文字 |
| Small | 11px | 400 | 标签 |

字体族：`"Noto Serif SC", "Source Han Serif SC", "STSong", "PingFang SC", "Microsoft YaHei", serif`

## 间距

| 级别 | 值 | 用途 |
|------|-----|------|
| xs | 4px | 紧凑间距 |
| sm | 8px | 元素内边距 |
| md | 14px | 卡片内边距 |
| lg | 18px | 板块间距 |
| xl | 24px | 页面边距 |

## 圆角

| 级别 | 值 | 用途 |
|------|-----|------|
| sm | 8px | 按钮、标签 |
| md | 12px | 卡片 |
| lg | 18px | 弹窗、大容器 |
| full | 50% | 圆形元素 |

## 阴影（更柔和）

- 卡片：`0 2px 12px rgba(0, 0, 0, 0.05), 0 1px 3px rgba(0, 0, 0, 0.03)`
- 弹窗：`0 8px 32px rgba(0, 0, 0, 0.10)`
- 按钮悬停：`0 2px 8px rgba(107, 159, 212, 0.20)`
