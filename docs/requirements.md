# 需求文档 — 健身计划管理 App

## 项目概述

一款运行在安卓手机上的健身计划管理应用，兼具备忘录功能，帮助用户记录健身计划、心得感悟，并通过直观的时间轴查看每日安排。

## 目标用户

- 健身爱好者
- 需要计划管理 + 备忘录的人群
- 偏好简洁清新界面的用户

## 功能需求

### FR-1：三大核心板块

| 编号 | 板块 | 描述 |
|------|------|------|
| FR-1.1 | 日历封面页 | 月历视图，每天显示任务圆点摘要，点击展开当日计划详情 |
| FR-1.2 | 心得记录 | 卡片式备忘列表，支持新建/编辑/删除/搜索 |
| FR-1.3 | 横轴时间表 | 双层横向设计，外层日期滑动 + 内层时间段活动展示 |

### FR-2：日历封面页详细需求
- 月历网格，默认当月
- 每月天格内用彩色圆点表示任务
- 点击日期在日历下方展开详情（不跳转）
- 支持添加/删除计划事项
- 左右箭头切换月份

### FR-3：心得记录详细需求
- 卡片列表，时间倒序
- 每条含日期、标题、正文、心情表情
- 新建/编辑/删除功能
- 关键词搜索

### FR-4：横轴时间表详细需求
- 外层：日期卡片横向滑动（近7天）
- 内层：横向时间轴（06:00~23:00）
- 活动块含图标 + 标题，不同颜色区分
- 点击活动编辑，点击空白添加
- 「今天」快捷按钮

### FR-5：健身图标
- 14个健身房动作 SVG 图标
- 统一蓝白线条风格

## 非功能需求

### NFR-1：平台
- 运行于安卓手机浏览器
- 支持 PWA 安装到桌面

### NFR-2：性能
- 页面加载 < 2 秒
- 交互响应流畅

### NFR-3：数据
- 浏览器本地存储（IndexedDB）
- 预留云端同步接口

### NFR-4：风格
- 主色调：蓝白
- 风格：清新文艺、简洁干净
