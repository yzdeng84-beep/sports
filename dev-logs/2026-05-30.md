# 开发日志 — 2026-05-30

## 完成内容

### 1. 消费图表（日/周/月）
- 新增 `js/chart.js` — 纯 SVG 图表模块（`ChartModule`）
- 日视图：最近7天柱形图（收入/支出并列柱）
- 周视图：最近4周柱形图（默认选中）
- 月视图：最近6个月折线图（收入/支出双线+数据点）
- 图表嵌入时间轴页面，位于记账汇总和记账列表之间
- 新增 `DB.expenses.getByDateRange()` / `getByMonth()` / `getByWeek()` 查询方法

### 2. 时间轴改为 SVG 水平视图
- 水平时间轴 SVG（viewBox "0 0 960 180"，覆盖 06:00-22:00）
- 整点网格线 + 时间标签
- 任务块按时间定位，自动车道分配避免重叠
- 缩放按钮（+/-）控制 1x~5x 缩放
- 双指捏合手势支持
- 缩放 >1x 时允许水平滚动
- 今日时间红色虚线指示器
- 点击任务块调出编辑弹窗

### 3. 版本号 1.0
- `manifest.json` 新增 `"version": "1.0"`
- `app/build.gradle`：`versionCode` 1→2，`versionName` 保持 "1.0"
- `sw.js`：`CACHE_NAME` → `cadence-v3`，新增 `chart.js` 到缓存列表

### 4. 默认标题
- 新建任务时标题默认显示所选图标名称（如"深蹲"、"跑步"）
- 切换图标时自动更新标题（仅当标题为空或为已知图标名时覆盖）
- 用户手动输入的自定义标题不会被覆盖

### 5. 手动更换背景（预设主题 + 导入图片）
- 标题栏右侧新增 🎨 按钮，弹出主题选择器
- 5套预设主题：默认蓝白、暖橙、青绿、素灰、暗夜
- 支持从相册导入图片作为背景（base64 存 localStorage）
- 背景渐变+点阵改为 CSS 变量驱动，支持图片覆盖
- 偏好持久化到 localStorage，刷新后保持
